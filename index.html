A motion recognition library for Android

Introduction

This library allows you to recognize short motions with your wearable device. Create and edit motions on the developer portal, and use the library to recognize motions in your app.

The first 1000 motion detections are free, once you utilize more than this please contact use for our commercial terms of use.

For commercial use of this library, contact us.

Features

  • Compatible with any device with an accelerometer and gyroscope.
  • Motion simulator to quickly test your recorded motions.
  • Automatic data and network management. Don't have to worry about making requests to our servers for anything.

How to use

Add our bintray repo to your repository list in build.gradle. 

repositories {
    jcenter()
    maven { url 'http://dl.bintray.com/kiwiwearables/Kiwi-lib' }
}
Grab the AAR via Gradle: 
compile 'com.kiwiwearables:kiwilib:0.3.1'

This library makes use of the internet to communicate with our web APIs so you'll need to specify internet permission in your manifest.

<uses-permission android:name="android.permission.INTERNET" />

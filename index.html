A motion recognition library for Android

Introduction

This library allows you to recognize short motions with your wearable device. Let the users have a completely different interactive experience with your app.

Features

  • Compatible with any internet-of-things device with an accelerometer and gyroscope (supports ms2 & rad/s, and G & deg/s out of the box).
  • Create and edit custom motions to recognize with our detection engine, on the web and the Android library.
  • Time-saving tools to test created motions
  • Automatic data and network management. Don't have to worry about making requests to our servers for anything.

Download

Add our bintray repo to your repository list in build.gradle. 

repositories {
    jcenter()
    maven { url 'http://dl.bintray.com/kiwiwearables/Kiwi-lib' }
}
Grab the AAR via Gradle: 
compile 'com.kiwiwearables:kiwilib:0.3.1b'
A sample app utilizing this library and getting sensor data from Android Wear is available on GitHub.
